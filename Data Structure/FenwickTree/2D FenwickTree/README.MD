# 2차원 펜윅 트리

## 개요
* 펜윅트리를 2차원으로 확장한 자료구조.

## 복잡도
#### 시간: 
* 구축: O(NMlogNM)
* 질의, 갱신: O(logNM)

#### 공간:
* O(NM)
###### N, M: 2차원 배열의 넓이와 높이

## 특징
* 포함 배제의 원리에 의해서 구간합을 구함.
* 하나의 값에 대한 질의만을 처리하는 대신 구간 갱신을 O(logNM)만에 수행 할 수있다.
